Block vs. Inline Elements - Divs and Spans

排版

<div>: group elements together

if grouped, we can use CSS to 把他们一起操作，比如一个带图片和文字的按钮

<span>: if spaned, we can use CSS to decorate them separatly

span和div两个标记都可以独立出区块，这一点没有很大的区别。 span和div区别在于，div是一个块级元素，它包含的元素会自动换行。 而span是行内元素，在它的前后不会换行。 span没有结构上的意义，只是单纯的应用样式，其他元素不适合时，就可以使用span元素。
